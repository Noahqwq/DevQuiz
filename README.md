# DevQuiz

Lernprojekt.

DevQuiz ist ein kleines Konsolenquiz für angehende Fachinformatiker:innen. Das Programm stellt Fragen zu C#, Git, GitHub, 
Datenbanken oder allgemeinen IT-Grundlagen. Am Ende wird ein Ergebnis angezeigt.
